## 内置 Event

| 事件编码            | 事件描述           |
|-----------------|----------------|
| serviceStatus   | driver-box服务状态 |
| deviceStatus    | 设备离线事件         |
| addDevice       | 添加设备           |
| willExportTo    | 即将执行 ExportTo  |
| deviceDiscover  | 设备发现           |
| linkEdgeTrigger | 触发场景联动         |
